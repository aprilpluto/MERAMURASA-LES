# Slot Aset Meramu Rasa

Letakkan file gambar Anda di folder ini:

| File | Ukuran disarankan | Keterangan |
|------|-------------------|------------|
| `logo.png` | 200×200 px (transparan PNG) | Logo Meramu Rasa — tampil di navbar |
| `mentor.jpg` | 800×800 px (persegi) | Foto profil Ardi Kamal Karima |

File PNG/JPG akan tampil di navbar dan bagian mentor (Next.js: `/assets/...`, HTML statis: `public/assets/...`).

## Contoh path setelah upload

```
public/assets/logo.png
public/assets/mentor.jpg
```
