# Meramu Rasa

Les cipta puisi daring — Next.js 15 + Tailwind. Pendaftaran les dan pengiriman karya dilakukan langsung via **WhatsApp**.

## Menjalankan

```bash
npm install
npm run dev
```

Buka **http://localhost:3000** (bukan `meramu-rasa.html` di browser — file HTML itu versi lama).

## Deploy Vercel

Tidak ada variabel lingkungan wajib. Nomor WhatsApp dikonfigurasi di `lib/constants.ts`.

## Aset

Letakkan di `public/assets/`:

- `logo.png` — navbar
- `mentor.jpg` — foto mentor
